This contains all necessary files for the p-body quantification assay. Quantification was done in Fiji, analysis in R Studio.

For a detailed description check the WorkFlow word file.

The .jim, .labeling and .classifier files were used during the quantification process.
Analyz.txt files indicate all macros used for the quantification in Fiji.
R code for statistical analysis via two way ANOVA and creating of images.
